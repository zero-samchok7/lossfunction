## lesson1.html — 직선 그리기와 오차

**학습 목표:** 산점도에 직접 직선을 그어보고, 직선과 데이터 사이의 오차를 직관적으로 이해한다.

### 화면 구성

| 영역 | 내용 |
|------|------|
| 좌측 패널 | 데이터 입력 테이블 (x, y), 샘플 생성 / 초기화 버튼 |
| 중앙 캔버스 | 산점도 — 점 추가 / 직선 그리기 / 직선 이동 |
| 우측 패널 (토글) | 직선별 예측값·오차 표, 오차 통계 4종 |

### 조작 방법

| 조작 | 설명 |
|------|------|
| **점 모드** | 캔버스 클릭으로 데이터 포인트 추가 |
| **직선 모드** | 빈 곳 클릭 → 두 번째 클릭으로 직선 생성 |
| **끝점 드래그** | 직선 끝점을 드래그해 기울기·위치 조절 (반경 14 px) |
| **직선 본체 드래그** | 직선 위를 드래그해 직선 전체 이동 (거리 10 px 이내) |
| **▶ 오차 버튼** | 우측 패널 열기·닫기 토글 |
| **예측값 보기** | 우측 테이블에 예측값(ŷ) 열 표시 |
| **오차 보기** | 우측 테이블에 오차(y−ŷ) 열 표시 |
| **자료 버튼** | 오차의 합·절댓값의 합·제곱근의 합·제곱의 합 (각 평균 포함) 표시 |

**터치(패드) 지원:** 끝점 드래그, 직선 본체 드래그 모두 터치 이벤트 처리됨.

## lesson2.html — 기울기 조절과 MSE 비교

**학습 목표:** 두 직선 y = a₁x, y = a₂x의 MSE를 비교하며 더 좋은 예측 직선의 의미를 파악한다.

### 화면 구성

| 영역 | 내용 |
|------|------|
| 좌측 패널 | 데이터 테이블, a₁·a₂ 슬라이더 |
| 중앙 캔버스 | 산점도 (두 직선 + 잔차선 표시), 범위 0~30 |
| 우측 패널 (토글) | MSE 전개식, MSE 비교 셀, 최적 추세선 박스 |

### 조작 방법

| 조작 | 설명 |
|------|------|
| a₁·a₂ 슬라이더 | 기울기 조절 → 캔버스·MSE 실시간 갱신 |
| **▶ 손실함수 버튼** | 우측 패널 열기·닫기 토글 |
| **최적 추세선 표시** | MSE 최솟값의 기울기(a*) 초록 직선으로 표시 |

### 수식 (KaTeX 렌더링)

우측 패널의 MSE 전개식은 슬라이더 변경·샘플 생성 때마다 실시간 갱신됩니다.

```
MSE(a) = [(y₁ - a·x₁)² + (y₂ - a·x₂)² + ···] ÷ n
```

## lesson3.html — 손실함수와 최솟값

**학습 목표:** MSE(a)를 이차함수로 전개하고, 포물선 그래프에서 최솟값을 찾는 과정을 단계별로 살펴본다.

### 화면 구성

| 영역 | 내용 |
|------|------|
| 좌측 패널 | 산점도 캔버스, 데이터 테이블, 기울기 a 슬라이더, MSE(a) 수치 |
| 중앙 캔버스 | MSE(a) 포물선 + 현재 a 위치 (빨간 점) |
| 우측 패널 (토글) | 손실함수 3단계 전개, 최솟값 찾기 버튼·결과 |

### 조작 방법

| 조작 | 설명 |
|------|------|
| a 슬라이더 | 포물선 위 현재 점 이동, 캔버스 클릭도 가능 |
| **▶ 손실함수 버튼** | 우측 패널 열기·닫기 토글 |
| **다음 단계 →** | 손실함수 전개 단계를 1→2→3 순으로 표시 |
| **최솟값 찾기** | 포물선 최저점으로 애니메이션 이동 |

### 수식 3단계 전개 (KaTeX 렌더링)

| 단계 | 수식 |
|------|------|
| 1단계 | MSE(a) = (1/n)·[(y₁−ax₁)² + (y₂−ax₂)² + ··· + (yₙ−axₙ)²] |
| 2단계 | = (Σxᵢ²/n)·a² − (2Σxᵢyᵢ/n)·a + Σyᵢ²/n |
| 3단계 | = **A**·a² − **B**·a + **C** (A, B, C: 데이터에서 계산한 실제 값, 보라색 강조) |
| 꼭짓점 | a* = B / (2A) |

- 포물선 위 ★ 표시와 a* 레이블은 **최솟값 찾기 버튼을 누른 후에만** 나타납니다.

## lesson4a.html — 손실함수의 이차곡면

**학습 목표:** y = ax + b의 MSE(a, b)가 2변수 이차함수(그릇 모양 곡면)임을 3D 시각화로 이해한다.

### 화면 구성 (3단 수평 레이아웃)

| 영역 | 폭 | 내용 |
|------|----|------|
| 1단 (데이터) | 240 px | 데이터 입력 테이블, 샘플 생성·초기화 버튼 |
| 2단 (수식·통계) | 240 px | MSE(a,b) 정의 및 전개 계수, 현재 점(a·b 슬라이더), 최솟값 |
| 3단 (3D 곡면) | 나머지 | 3D 손실함수 곡면 캔버스 |

### 조작 방법

| 조작 | 설명 |
|------|------|
| a 슬라이더 | 3D 곡면 위 빨간 점의 a 값 조절 (−5 ~ 5) |
| b 슬라이더 | 3D 곡면 위 빨간 점의 b 값 조절 (−10 ~ 10) |
| **3D 곡면 드래그** | 자유 회전 (초기 각도: 그릇 모양, 최솟값이 아래) |
| **★ 최솟값 찾기** | 현재 점이 최솟값(a*, b*)으로 애니메이션 이동 |

### 수식 (KaTeX 렌더링)

```
MSE(a,b) = (1/n) · Σ(yᵢ − axᵢ − b)²
         = A·a² + B·b² + C·ab − D·a − E·b + F
```

### 3D 곡면 표현

- 투시 투영 + 페인터 알고리즘으로 그릇 모양 렌더링
- X축: a (빨강), Y축: MSE (파랑), Z축: b (초록) 화살표·레이블 표시
- 그릇 최저점 = 최솟값 (a*, b*)

## lesson5.html — 경사하강법 종합

**학습 목표:** MSE(a) 포물선 위에서 경사하강법이 최솟값을 찾아가는 과정을 단계별로 시각화한다.

### 화면 구성

| 영역 | 폭 | 내용 |
|------|----|------|
| 좌측 패널 | 240 px | 데이터 테이블, 산점도 캔버스 |
| 중앙 패널 | flex:1 | 컨트롤 & 수치 — 실행 버튼, 초기 설정, 공식, 현재 수치, 단계별 기록 |
| 우측 패널 | 360 px | MSE 포물선 그래프 (1사분면, 원점 포함) |

### 조작 방법

| 조작 | 설명 |
|------|------|
| 초깃값 a₀ | 경사하강법 시작점 수동 입력 |
| 학습률 α | 슬라이더 0.001 ~ 0.5 (소수점 3자리 표시) |
| 반복 횟수 n | 슬라이더 1 ~ 30 |
| **▶ 실행** | 애니메이션 실행 (속도 슬라이더로 조절) |
| **⏸ 일시정지** | 현재 단계에서 정지 |
| **실행단계 슬라이더** | 특정 단계로 이동해 포물선 위 위치 확인 |
| **↺ 초기화** | 경사하강법 상태만 초기화 (데이터 유지) |

### 수식 (KaTeX 렌더링)

```
a_new = a_old − α × (접선의 기울기)
```

공식 아래에는 현재 단계의 실제 수치가 자동으로 업데이트됩니다.

### 포물선 그래프 표시 요소

| 요소 | 설명 |
|------|------|
| 파란 포물선 | MSE(a) 전체 곡선 |
| 빨간 ★ | 최솟값 위치 (a*) |
| 파란 점 | 현재 단계의 (a, MSE) |
| 빨간 접선 | 현재 단계의 접선 기울기 시각화 |
| 파란 경로 | 이전 단계들의 이동 궤적 |

**수렴 조건:** `|접선의 기울기| < 0.001` 이면 수렴 판정 후 자동 정지.  
그래프는 항상 **1사분면(원점 포함)** 으로 표시됩니다.
