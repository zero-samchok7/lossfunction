### lesson1.html — 직선 그리기와 오차

**학습 목표:** 산점도에 직접 직선을 그어보고, 직선과 데이터 사이의 오차를 직관적으로 이해한다.

**화면 구성 (좌 · 중 · 우)**

| 영역 | 내용 |
|------|------|
| 좌측 | 데이터 입력 테이블 (x, y), 샘플 생성 / 초기화 버튼 |
| 중앙 | 산점도 캔버스 — 점 추가 / 직선 그리기 / 직선 이동 |
| 우측 패널 | 직선별 예측값·오차 표, 자료 통계 (오차의 합·평균 4종) |

**주요 조작**
- **점 모드**: 캔버스 클릭으로 데이터 포인트 추가
- **직선 모드**: 두 번 클릭으로 직선 생성 → 끝점 드래그로 기울기/위치 조절 → 직선 본체 드래그로 전체 이동
- 터치(패드) 지원
- 우측 패널 "▶ 오차" 버튼으로 예측값/오차 열 표시 전환
- "자료" 버튼으로 오차의 합·절댓값의 합·제곱근의 합·제곱의 합 (각 평균 포함) 표시

### lesson2.html — 기울기 조절과 MSE 비교

**학습 목표:** 두 직선 y = a₁x, y = a₂x의 MSE를 비교하며 더 좋은 예측 직선의 의미를 파악한다.

**화면 구성**

| 영역 | 내용 |
|------|------|
| 좌측 | 데이터 테이블, a₁·a₂ 슬라이더 |
| 중앙 | 산점도 캔버스 (두 직선 + 잔차선 표시), 범위 0~30 |
| 우측 패널 | MSE 수식 전개, MSE 비교 셀 (승자 강조), 최적 추세선 |

**주요 조작**
- 슬라이더로 a₁(빨강), a₂(파랑) 조절 → 실시간 MSE 갱신
- "최적 추세선 표시" 버튼으로 MSE 최솟값의 기울기 확인
- 샘플 생성 범위: x ∈ [2, 16], 그래프 범위: 0~30

### lesson3.html — 손실함수와 최솟값

**학습 목표:** MSE(a)를 이차함수로 전개하고, 포물선 그래프에서 최솟값을 찾는 과정을 단계별로 살펴본다.

**화면 구성**

| 영역 | 내용 |
|------|------|
| 좌측 | 데이터 테이블, a 슬라이더, MSE 수치 |
| 중앙 | MSE(a) 포물선 + 현재 a 위치 표시 |
| 우측 패널 | 손실함수 단계별 전개 (3단계), 최솟값 찾기 애니메이션 |

**주요 조작**
- a 슬라이더 → 포물선 위 현재 점 이동
- "다음 단계 →" 버튼으로 MSE 수식 전개 과정 순차 표시
- "최솟값 찾기" 버튼 → 포물선 최저점으로 애니메이션 이동

### lesson4a.html — 손실함수의 이차곡면

**학습 목표:** y = ax + b의 MSE(a, b)가 2변수 이차함수(그릇 모양 곡면)임을 3D 시각화로 이해한다.

**화면 구성 (1단 · 2단 · 3단)**

| 영역 | 내용 |
|------|------|
| 1단 (240px) | 데이터 입력 테이블 |
| 2단 (240px) | MSE 전개식 계수, 현재 점 수치 + a·b 슬라이더, 최솟값 |
| 3단 (나머지) | 3D 곡면 캔버스 (X=a, Y=MSE, Z=b 축 표시) |

**주요 조작**
- 3D 곡면 드래그로 자유 회전 (초기 각도: 그릇 모양, 최솟값이 아래 위치)
- 2단의 a·b 슬라이더 → 곡면 위 빨간 점 이동
- "★ 최솟값 찾기" 버튼 → 점이 최솟값으로 애니메이션 이동

### lesson5.html — 경사하강법 종합

**학습 목표:** MSE(a) 포물선 위에서 경사하강법이 최솟값을 찾아가는 과정을 단계별로 시각화한다.

**화면 구성 (좌 · 중 · 우)**

| 영역 | 내용 |
|------|------|
| 좌측 (240px) | 데이터 테이블 (행 추가 무제한), 산점도 |
| 중앙 (flex) | 컨트롤 & 수치 — 실행 버튼, 초기 설정, 공식, 현재 수치, 단계별 기록 |
| 우측 (360px) | MSE 포물선 그래프 (1사분면, 원점 포함) |

**주요 조작**

| 컨트롤 | 설명 |
|--------|------|
| 초깃값 a₀ | 경사하강법 시작점 |
| 학습률 α | 0.001 ~ 0.5, 소수점 3자리 표시 |
| 반복 횟수 n | 1 ~ 30 |
| ▶ 실행 | 애니메이션 실행 (속도 슬라이더로 조절) |
| ⏸ 일시정지 | 현재 단계에서 정지 |
| 실행단계 슬라이더 | 특정 단계로 이동하여 확인 |
| ↺ 초기화 | 경사하강법 상태만 초기화 (데이터 유지) |

**포물선 그래프 표시 요소**
- 파란색 포물선 MSE(a)
- 빨간 ★: 최솟값 위치 (a*)
- 파란 점: 현재 단계의 (a, MSE)
- 빨간 접선: 현재 단계의 접선 기울기
- 파란 이동 경로 (이전 단계들)
- 수렴 조건: |접선의 기울기| < 0.001

## 공통 사항

- **샘플 생성**: 모든 차시 10개 점 생성 (x ∈ [2, 16], 양의 기울기 무작위)
- **점 추가**: 빈 행에 입력하면 자동으로 데이터에 추가됨 (무제한)
- **뒤로 가기**: 각 페이지 상단 `← 목록` 버튼 → index.html
- **터치 지원**: lesson1 캔버스는 touchstart/touchmove/touchend 처리
