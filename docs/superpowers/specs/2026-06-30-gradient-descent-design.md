# 경사하강법 탐구 — 설계 문서

**날짜:** 2026-06-30  
**대상:** 고등학교 인공지능수학 (2022 교육과정 예측과 최적화 단원)  
**환경:** iPad 가로 모드

---

## 1. 개요

학생이 경사하강법의 원리를 직접 탐구할 수 있는 인터랙티브 웹페이지.  
기울기 a 하나만 최적화하는 1변수 경사하강법에 집중하며,  
데이터 보정(무게중심 → 원점 이동)을 통해 y = ax 형태를 수학적으로 정당화한다.

---

## 2. 핵심 수학 설계

### 데이터 보정 원리
- 원본 데이터 (xᵢ, yᵢ)의 무게중심: (x̄, ȳ)
- 보정 데이터: x'ᵢ = xᵢ − x̄, y'ᵢ = yᵢ − ȳ
- 보정 후 무게중심 = 원점 → y' = ax' 형태 성립 (b=0 수학적 정당)
- 원본 좌표 복원 시: y = ax + b, b = ȳ − a·x̄

### 손실함수 (MSE)
```
MSE(a) = (1/n) Σ(y'ᵢ − a·x'ᵢ)²
```
→ a에 대한 포물선 형태

### 경사하강법 업데이트 규칙
```
dMSE/da = (−2/n) Σx'ᵢ(y'ᵢ − a·x'ᵢ)

a_new = a_old − α × dMSE/da
```

### 수렴 조건
```
|dMSE/da| < 0.001 → 수렴 완료, 자동 정지
```

## 4. 패널별 상세 설계

### 4-1. 왼쪽 패널 — 데이터 & 산점도

**데이터 입력**
- x, y 입력 테이블 (최대 20행)
- [샘플 생성] 버튼: 10개 랜덤 데이터 자동 생성
- [초기화] 버튼: 전체 데이터 삭제

**산점도 (캔버스)**
- 무게중심 (x̄, ȳ) 항상 표시
- [원본 보기 ↔ 보정 보기] 전환 버튼
  - 원본 보기: 원본 좌표 산점도만 표시
    - 경사하강법 진행 중에는 y = ax + b 추세선도 표시
  - 보정 보기: 무게중심이 원점으로 이동하는 애니메이션 → y = ax 추세선 표시

**수치 표시**
- 추세선 방정식 실시간 표시
  - 보정 보기: y = ax (a값 실시간 업데이트)
  - 원본 보기: y = ax + b (a, b값 실시간 업데이트)
- 현재 MSE값

**연동**
- 경사하강법 진행 중 추세선 실시간 업데이트 (중간 패널과 동기화)

---

### 4-2. 중간 패널 — MSE 포물선 & 경사하강법 시각화

**좌표평면**
- x축: 기울기 a
- y축: MSE(a)
- MSE(a) 포물선 자동 생성 (데이터 보정 후)
- 최솟값 위치 별도 표시 (★)

**경사하강법 시각화**
- 현재 단계의 점 (포물선 위)
- 현재 점에서의 접선 표시
- 지나온 자취 (trail, 색상 그라데이션)
- 포물선 클릭 → 초깃값 a₀ 직접 설정

**컨트롤 (하단)**
- [▶ 실행] [⏸ 일시정지] [↺ 초기화] 버튼
- 애니메이션 속도 슬라이더
- 단계 슬라이더 (1 ~ 30): 원하는 단계로 직접 이동

---

### 4-3. 오른쪽 패널 — 컨트롤 & 수치

**초기 설정**
- 초깃값 a₀: 숫자 직접 입력 (포물선 클릭과 연동)
- 학습률 α: 슬라이더 (0.01 ~ 1.0)
- 반복 횟수 n: 슬라이더 (1 ~ 30)

**경사하강법 공식 실시간 표시**
```
a_new = a_old − α × (dMSE/da)
      = [현재 a값] − [α값] × [현재 기울기값]
      = [계산 결과]
```

**현재 단계 수치**
- 현재 단계 n
- 현재 a값
- 현재 MSE값
- 현재 접선 기울기 (dMSE/da)
- 수렴 완료 표시: |dMSE/da| < 0.001 시 자동 정지 + "수렴 완료" 표시

**단계별 기록 테이블**
- 컬럼: 단계(n) / a / MSE / 기울기(dMSE/da)
- 각 단계마다 행 누적

---

## 5. 데이터 흐름

```
데이터 입력
    ↓
무게중심 (x̄, ȳ) 계산
    ↓
보정 데이터 생성 (x'ᵢ, y'ᵢ)
    ↓
MSE(a) 포물선 계산 → 중간 패널 렌더링
    ↓
초깃값 a₀ 설정 (입력 or 클릭)
    ↓
경사하강법 실행 (매 단계):
  - dMSE/da 계산
  - a_new = a_old − α × dMSE/da
  - |dMSE/da| < 0.001 이면 수렴 정지
    ↓
왼쪽 패널 추세선 실시간 동기화
```
