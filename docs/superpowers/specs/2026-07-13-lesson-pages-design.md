## 1. 전체 구조

```
index.html          ← 5차시 허브 메인 (기존 허브 교체)
lesson1.html        ← 1차시: 추세선 직접 그리기
lesson2.html        ← 2차시: y=ax 직선 비교와 SSE
lesson3.html        ← 3차시: 손실함수와 최솟값 (개발 예정)
lesson4.html        ← 4차시: 경사하강법 (gradient.html 재활용)
lesson5.html        ← 5차시: 종합 (상세 미정)
```

**공통 규칙**
- 모든 페이지 상단 좌측에 `← 목록` 버튼 → `index.html`로 이동
- ES 모듈 없음, fetch 없음 — 더블클릭으로 바로 열림
- iPad landscape 최적화 (100dvh, overflow:hidden, 3패널 flex)

## 2. index.html — 허브 메인

- 기존 2카드 허브를 5차시 카드로 교체
- 카드 배열: 1~5차시 순서대로 가로 flex (wrap)
- 각 카드: 차시 번호, 제목, 한 줄 설명, [열기] 버튼
- 차시 흐름 표시: 1→2→3→4→5 (화살표 또는 진행 표시)

| 차시 | 제목 | 설명 |
|------|------|------|
| 1차시 | 추세선 직접 그리기 | 직접 직선을 그려 예측값과 오차를 확인합니다 |
| 2차시 | y=ax 직선 비교 | 두 직선의 오차제곱합을 비교하며 최적 직선을 탐색합니다 |
| 3차시 | 손실함수와 최솟값 | a에 따른 손실함수 그래프에서 최솟값을 찾습니다 |
| 4차시 | 경사하강법 | 경사를 따라 최솟값을 찾아가는 과정을 탐구합니다 |
| 5차시 | 종합 | (상세 미정) |

## 3. lesson1.html — 추세선 직접 그리기

### 3패널 레이아웃

**좌측 패널 (고정 폭)**
- x, y 데이터 입력 테이블
  - 범위: 0~100, 최대 15행
  - x, y 열 직접 입력
- [샘플 생성] 버튼: 랜덤 선형 데이터 10개 자동 생성
- [초기화] 버튼: 전체 데이터 삭제

**중간 패널 (flex 확장)**
- 좌표평면 캔버스
  - 데이터 범위에 따라 자동 줌 (auto-scale viewport)
  - 1사분면 위주 (x, y ≥ 0)
  - 격자선, 축 레이블
- 모드 전환 버튼: [점 추가 모드] / [직선 그리기 모드]
  - 점 추가 모드: 그래프 클릭 → 데이터 포인트 추가 (테이블에도 반영)
  - 직선 그리기 모드: 클릭 2점으로 직선 생성, 직선 이동·기울기 조정 가능
- 직선 여러 개 동시 표시 가능 (각 직선 방정식 레이블 표시)
- 우측 상단: ▶ 버튼 → 오른쪽 패널 열기/닫기

**우측 패널 (처음엔 숨김, ▶로 펼침)**
- 직선 선택 드롭다운 (직선이 여러 개일 때)
- 예측값·오차 표
  | x | 예측값(ŷ) | 오차(y−ŷ) |
  |---|-----------|-----------|
  | x₁ | — | — |
  | x₂ | — | — |
  - x값: 왼쪽 테이블에 입력된 x값들
  - 예측값 = 선택된 직선에서 해당 x의 y좌표
  - 오차 = 실제 y − 예측값
- [예측값 보기] 버튼: 예측값 열 표시/숨김
- [오차 보기] 버튼: 오차 열 표시/숨김

## 4. lesson2.html — y=ax 직선 비교와 SSE

### 3패널 레이아웃

**좌측 패널**
- x, y 데이터 입력 테이블
  - 범위: 0~20, 최대 15행
- [샘플 생성] [초기화] 버튼

**중간 패널**
- 좌표평면 (0~20 고정 범위)
- y=a₁x 직선 (색상 1) + y=a₂x 직선 (색상 2) 동시 표시
- 각 직선 방정식 레이블 (y = 1.5x 등)
- 잔차선(오차) 시각적 표시 (선택적)

**우측 패널**
- a₁ 슬라이더 (0~5, 소수점 1자리) + 수치 표시
- a₂ 슬라이더 (0~5, 소수점 1자리) + 수치 표시
- 손실함수 원래식 표시 (슬라이더 변경 시 실시간 갱신)
  - a₁ 기준: `(y₁−a₁x₁)² + (y₂−a₁x₂)² + ... = [SSE₁값]`
  - a₂ 기준: `(y₁−a₂x₁)² + (y₂−a₂x₂)² + ... = [SSE₂값]`
- SSE 비교: 두 값을 나란히 수치 표시 (어느 쪽이 더 작은지 강조)
- [최적 추세선 표시] 버튼 → 최소 SSE인 y=ax 직선 등장 + 최적 a값 표시

## 5. lesson3.html — 손실함수와 최솟값

*(gradient.html의 수학 엔진 활용, UI 새로 구성)*

**좌측 패널**: 데이터 입력 + 산점도  
**중간 패널**: a값에 따른 MSE(a) 포물선 그래프  
- x축: 기울기 a
- y축: MSE(a) = (1/n)Σ(yᵢ−axᵢ)²
- 포물선 위 클릭 → a값 선택
- 최솟값 위치 ★ 표시 + a* 레이블

**우측 패널**: 손실함수 식, 현재 a값, MSE값, 최적 a 표시
